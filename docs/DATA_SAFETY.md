# Google Play Data Safety Draft

## Data collection

PowerMate AI v1 does not collect or transmit personal data to a remote server.

## Local-only data

The app stores charging history locally on the device:

- Charging session time
- Battery percentage changes
- Estimated current/wattage when supported
- Voltage/temperature when available
- Diagnostic scores

## Data sharing

No data is shared with third parties in v1.

## Security practices

- No account required
- No cloud upload in v1
- User can clear history
- Permissions are requested only when needed
