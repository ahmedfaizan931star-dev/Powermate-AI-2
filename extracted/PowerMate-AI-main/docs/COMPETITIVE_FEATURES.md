# Competitive Feature Expansion

PowerMate AI should cover the common features found in AmpereFlow, Ampere, AccuBattery, and Battery Guru while staying original, privacy-first, and free-feature focused.

## AmpereFlow-Style Coverage

- Live ampere reading
- Live wattage reading
- Accurate battery percentage
- Charging animation
- Fast charging detection
- Always-On Display style charging screen
- Battery health monitoring
- Voltage reading
- Temperature reading
- Customizable battery speedometer
- Battery level alerts
- Widgets
- Customizable AOD metrics
- More AOD styles
- AOD timeout
- Media controls concept on AOD
- Torch/camera shortcut concept on AOD
- Custom notification format
- Charging history tab
- Option to show wattage instead of ampere

## Competitor-Inspired Improvements

- Actual capacity estimate concept
- Battery wear per cycle concept
- Screen-on vs screen-off charge estimate
- Charger/cable comparison
- Charging stability score
- Best charger ranking
- Slowest charger warning
- Overheat alerts
- Battery care reminders
- Smart Charging Coach
- Offline-first local history

## PowerMate AI Differentiators

- All core tools included in v1
- No feature locks in core flows
- No account required
- Offline-first
- Material 3 Android-native UI
- Honest fallback for unsupported sensors
- Safe settings shortcuts instead of risky system hacks
