# PowerMate AI PRD

## Product summary

PowerMate AI is an offline-first Android battery monitor, charging diagnostics, charging history, widgets, smart alerts, and AOD-style charging display app.

## Product goal

Create a cleaner and more generous alternative to battery monitor and charging AOD apps. All core tools stay free.

## Core features

- Live ampere/current reading
- Live wattage estimate
- Voltage and temperature
- Fast/slow charging detection
- Battery health status
- Charger quality score
- Cable quality score
- Charging stability score
- Charging history
- AOD-style charging display
- AOD timeout, night dim and burn-in protection
- Battery alerts at 80%, 90%, full, overheat, slow charging and disconnect
- Widgets
- Offline-first privacy

## MVP success

- App opens without crash
- Dashboard updates every second while visible
- Diagnostic flow completes and saves result
- AOD screen opens and displays battery data
- Widget shows battery percentage
- GitHub Actions builds debug APK, release APK and release AAB
