# PowerMate AI Privacy Policy

PowerMate AI is designed as an offline-first battery utility.

## Data collected

PowerMate AI reads battery and charging information available from Android system APIs, such as battery level, charging state, voltage, temperature, current when supported, and charging history created inside the app.

## Data storage

Charging history is stored locally on the user's device. The app does not require an account and does not upload charging history to a server in version 1.

## Permissions

- Notifications: used for battery alerts such as 80%, 90%, full charge, slow charging, charger disconnect, and overheat warnings.
- Foreground service: used only for active monitoring sessions or AOD-style charging display when enabled.
- Overlay permission: requested only if the user enables overlay-style charging display.
- Boot completed: used to restore enabled alert behavior after device restart.

## Third-party sharing

PowerMate AI does not sell user data. Version 1 does not include cloud sync, login, or remote analytics.

## User controls

Users can clear local history from Settings. Users can disable alerts and AOD-style display anytime.
