# Smart Charging Coach

PowerMate AI includes a Smart Charging Coach that gives safe charging-speed suggestions without using root access or private Android APIs.

## Purpose

The coach helps users improve charging speed and reduce heat by suggesting user-controlled optimizations:

- Open Internet controls to disable Wi-Fi/mobile data/hotspot if not needed
- Open Bluetooth settings to disable Bluetooth if unused
- Open Location settings to disable GPS/location scanning if unused
- Open Display settings to reduce brightness or enable adaptive brightness
- Open Battery Saver settings during slow charging
- Open NFC settings if NFC is not needed
- Recommend removing a thick case if the phone is hot
- Recommend running charger/cable diagnostic when slow charging is detected

## Android Shortcut Policy

Modern Android versions usually do not allow normal apps to directly toggle Wi-Fi, Bluetooth, mobile data, GPS, or similar system controls silently. PowerMate AI therefore opens official Android settings screens or settings panels and keeps the user in control.
